# Aplikasi Biodata Siswa.js

## Struktur Folder

biodata-siswa
 ┣ config                   -> Berisi konstanta global (nama aplikasi)
 ┣ data                     -> Berisi data siswa
 ┣ logic                    -> File utama untuk menjalankan program
 ┣ app.js                   -> Dokumentasi proyek
 ┗ Readme.md

 ## Cara menjalankan

 1. Buka terminal
 2. Jalankan perintah:
 node app.js
